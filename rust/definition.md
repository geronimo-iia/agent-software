# Rust

Rust development standards covering project structure, tooling, configuration, and best practices for building safe, performant, and maintainable Rust software.

This topic covers:

- **Project Structure**: Standard directory layouts and workspace organization
- **Development Tools**: Essential tooling for Rust development (cargo, clippy, rustfmt, etc.)
- **Configuration**: Project configuration files and settings
- **Code Quality**: Linting, formatting, and static analysis standards
- **Testing**: Testing frameworks and practices
- **Packaging**: Crate distribution and dependency management

These standards ensure consistent, maintainable Rust projects with modern tooling and best practices.
