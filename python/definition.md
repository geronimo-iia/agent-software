Python development standards covering project structure, tooling, configuration, and best practices for building maintainable Python software.

This topic covers:

- **Project Structure**: Standard directory layouts and organization patterns
- **Development Tools**: Essential tooling for Python development (uv, ruff, pytest, etc.)
- **Configuration**: Project configuration files and settings
- **Code Quality**: Linting, formatting, and static analysis standards
- **Testing**: Testing frameworks and practices
- **Packaging**: Distribution and dependency management

These standards ensure consistent, maintainable Python projects with modern tooling and best practices.
