# Project Management Standards Auto-Activation

## Activation Triggers

When working with:
- Files named `roadmap.md`, `ROADMAP.md`, or `docs/roadmap.md`
- Files in `docs/archive/` directories (phase archival)
- Planning discussions mentioning "phase", "roadmap", "milestone", or "release"
- Project structure or documentation organization tasks

## Auto-Include Content

Include the following agent-software standards as context:
- `agent-software/project-management/definition.md`
- `agent-software/project-management/roadmap-documentation.md`

## Purpose

Ensures consistent roadmap structure, phase documentation, and archival patterns across all projects without manual reference lookup.

## Scope

Applies to all projects in the workspace that use phase-based development or maintain roadmap documentation.
